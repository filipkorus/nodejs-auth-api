# nodejs-auth-api

# About
Authentication and authorization token-based API written in Node.js. It can be used with whatever frontend application which requires authentication.

## Requirements
- Node.js
- SQL Database

## Project setup
- run `npm install` command to install node dependencies
- rename `example.env` file to `.env`
- set all credentials in `.env` file

### Runs in developer environment
```
npm run dev
```

### Compiles TypeScript
```
npm run build
```

### Starts compiled version
```
npm run start
```
